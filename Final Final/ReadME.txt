Shaft only contient les resultats des tests ou il n'y avait que la tige 
et shaft rotor contient ceux ayant la tige et le disque inertiel au milieu.

Ramping speed test contient les résultats des tests ou la vitesse augmente graduellement pour les spectrograms (waterfallo plots)

Continous speed tests contients les tests ou la vitesse est maintenu

Impact Test contient les resultats des tests servant a determiner la 1ere frequence naturelle de l'arbre


Les test misaligned ont été fait avec un décalage parallèle de 18mm au niveau du bearing supérieur.
